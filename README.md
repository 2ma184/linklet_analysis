# linklet_analysis
このレポジトリはlinkletファイルの解析のためのpythonライブラリーです。\
linklet：Emulsionを名古屋大学F研にあるHST-2（HTS-1）でスキャンし、二枚を接続したファイル

これをpullすればfrom linklet_analysis.<使いたい関数のファイル> import <使いたい関数>を書くだけで関数が使えます。\
pullした場所をubuntuなら~/.bashrcにexport PYTHONPATH=$HOME/<pullした場所>:$PYTHONPATHを書き込んでsource ~/.bashrcするとどこでも使えます。
